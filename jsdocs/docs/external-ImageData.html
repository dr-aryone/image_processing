ImageData

The built in image data interface; represents the underlying pixel data of an area of a <canvas> element.

Note: Below is a limited summary of the most commonly used properties and methods of ImageData in this package.
Properties:
Name Type Description
data Uint8ClampedArray Effectively a 1-dimensional array of pixel data in the format [R, G, B, A, R, G, B, A, ...]
width Number The width of the image, i.e. the number of pixels per row
height Number The height of the image, i.e. the number of rows in the image
