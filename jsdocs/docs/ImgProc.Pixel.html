Class: Pixel

ImgProc.Pixel

new Pixel(ropt, gopt, bopt, aopt)

A class to store color and alpha data for a single pixel. The Pixel constructor accepts five forms, ranging from 0 to 4 arguments:
new Pixel()rgba(0,0,0,255)
new Pixel(n)rgba(n,n,n,255)
new Pixel(n, a)rgba(n,n,n,a)
new Pixel(r, g, b)rgba(r,g,b,255)
new Pixel(r, g, b, a)  rgba(r,g,b,a)

Parameters:
Name Type Attributes Default Description
r Number <optional>
 0 red color channel value
g Number <optional>
 0 green color channel value
b Number <optional>
 0 blue color channel value
a Number <optional>
 255 alpha channel value
